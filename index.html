Flood Risk Mapping in the Volta Region, Ghana

Research Period: May 2023 – August 2023

Tools & Technologies: Google Earth Engine, Remote Sensing, OGC Standards

Project Overview

Flooding is a significant hazard in many regions, including Ghana's Volta region. This project leverages remote sensing datasets and terrain analysis to assess flood susceptibility. The workflow integrates multiple environmental factors known to influence flooding risk.

Data Sources

  • ESA WorldCover 2020: Global land cover classification for LULC data
  • SRTM Global 1 Arc-Second DEM: Elevation data for deriving slope
  • MERIT Hydro Dataset: Height Above Nearest Drainage (HAND)
  • JRC Global Surface Water: Historical surface water for flood frequency

Methodology

  1. Land Cover Reclassification: Remapped ESA WorldCover into 4 flood susceptibility classes
  2. Elevation Reclassification: Classified elevation ranges (lower = higher susceptibility)
  3. Slope Analysis: Flatter slopes indicate higher flooding risk
  4. Distance to Water: Calculated proximity to water bodies
  5. HAND Analysis: Lower HAND values = higher flood risk
  6. Flood Frequency: Calculated from 2000-2024 monthly water data
  7. Weighted Overlay: Combined all layers with scientifically-derived weights

Weighting Model

  • Flood Frequency: 2.5
  • Land Cover: 2.0
  • Elevation: 2.5
  • Slope: 2.5
  • Distance to Open Water: 1.5
  • HAND: 1.5

Impact: This model provides local authorities with data-driven flood risk assessments for emergency planning and resource allocation.

NDVI Calculation Using Sentinel-2 Imagery in Google Earth Engine

Tools & Technologies: Google Earth Engine, Sentinel-2, Remote Sensing

Project Overview

NDVI (Normalized Difference Vegetation Index) is calculated from Sentinel-2 satellite imagery using Google Earth Engine. The workflow includes cloud masking, compositing images over a date range, and visualizing both true color composites and NDVI layers.

What is NDVI?

NDVI stands for Normalized Difference Vegetation Index. It is a widely used remote sensing index that measures the presence and condition of green vegetation on Earth's surface.

  • Values close to +1 (0.8 - 1): Dense, healthy green vegetation
  • Values near 0: Bare soil, urban areas, or sparse vegetation
  • Negative values: Water, clouds, or snow

NDVI Formula

NDVI = (NIR - Red) / (NIR + Red)

Where NIR = Near Infrared band (B8) and Red = Red visible band (B4)

Applications

  • Agriculture: Monitoring crop health and detecting stress
  • Forestry: Assessing forest cover and biomass
  • Environmental Monitoring: Tracking droughts and desertification
  • Urban Planning: Assessing green spaces and vegetation
  • Climate Studies: Understanding vegetation responses to climate change

Impact: This analysis supports evidence-based environmental management and agricultural monitoring across Ghana.

Watershed Per Outlet Analysis: Sawla-Tuna-Kalba District, Ghana

Project Aim: Estimate total drainage area (watershed) contributing to each outlet

Project Overview

This project focuses on delineating watershed areas for various outlets within the Sawla-Tuna-Kalba District. A watershed is an area of land that drains all rainfall and streams to a common outlet. Understanding watershed boundaries is crucial for water resource management, flood risk assessment, and sustainable environmental planning.

Data & Methods

Data Source: SRTM Digital Elevation Model (DEM)

Methodology Steps:

  1. DEM Fill: Remove spurious depressions
  2. Flow Direction: Calculate water flow from each cell
  3. Flow Accumulation: Identify stream networks
  4. Reclassify: Extract river channels using flow accumulation threshold
  5. Outlet Creation: Generate outlets along the stream network
  6. Watershed Delineation: Process watersheds per outlet

Applications

  • Water Resource Allocation: Efficient distribution among communities
  • Flood Risk Management: Predicting flood-prone areas
  • Sustainable Water Management: Long-term water availability planning

Impact: This analysis enables informed water resource decisions and disaster preparedness for the district.

Image Segmentation with SAMGeo (GeoAI)

Project Goal: Segment satellite imagery to identify key campus landmarks

Project Overview

This project applies GeoAI to segment an image showing the Sam Jonah Library, G-Block, and Lecture Theater at the University of Cape Coast, Ghana. Using advanced image segmentation techniques, we accurately identify and separate these landmarks from satellite/aerial imagery.

What is Image Segmentation?

Image segmentation divides satellite or aerial images into meaningful groups of pixels (segments). This simplifies the image, making it easier to identify and analyze specific objects. Segmentation relies on characteristics such as size, texture, and shape to distinguish different regions.

About SAMGeo

SAMGeo is a Python library designed to adapt Meta AI's Segment Anything Model (SAM) for remote sensing. It is built on Vision Transformers (ViTs), which use attention mechanisms to concentrate on relevant image parts while ignoring less important details.

Key Feature: Zero-Shot Learning

One of SAMGeo's most impressive features is automatic segmentation without additional training. The pre-trained models (vit_h, vit_l, vit_b) are already familiar with segmenting diverse images, enabling accurate results on new, unseen remote sensing data.

How It Works

  1. Input satellite/aerial image of UCC campus
  2. Initialize SAMGeo with a pre-trained Vision Transformer model
  3. SAMGeo processes and segments the image into meaningful regions
  4. Results highlight landmarks (Library, G-Block, Lecture Theater)

Impact: This zero-shot learning approach enables rapid object detection and classification in remote sensing without specialized training data.

Flood Prediction Using U-Net Model: Akosombo Dam Spillage 2023

Focus: Deep learning for automatic flood area prediction

Project Overview

This project applies deep learning to automatically segment flooded regions from satellite/aerial images. Accurate flood mapping is critical for emergency response and resource allocation. The case study focuses on the Akosombo Dam spillage event in Ghana (2023).

Dataset & Preprocessing

Dataset Source: Kaggle Flood Segmentation Dataset with labeled flood images and ground truth masks

Data Preprocessing:

  • Normalized and resized images to consistent dimensions
  • Data augmentation (rotations, flips, brightness adjustments)
  • Training/Testing split: 70% / 30%

Model Architecture: U-Net

U-Net is a powerful CNN specifically designed for image segmentation with an encoder-decoder framework:

  • Encoder (Contracting Path): Extracts hierarchical features through convolution and max-pooling
  • Decoder (Expanding Path): Restores spatial resolution through upsampling
  • Skip Connections: Preserve spatial information and fine details like flood boundaries
  • Dropout Layers: Reduce overfitting and improve generalization

Model Performance

  • Test Accuracy: 87% - Percentage of correctly predicted pixels
  • Intersection Over Union (IoU): 72% - Agreement between predicted and actual flood extents
  • Test Loss: 30% - Acceptable range for this complex task

Explainability with Grad-CAM

To enhance model interpretability, Gradient-weighted Class Activation Mapping (Grad-CAM) produces heatmaps highlighting regions most influential in predictions. Red areas show where the model focused to identify flooded zones, increasing confidence in predictions.

Tools & Frameworks

  • TensorFlow: Building and training the U-Net model
  • Keras API: Simplified model construction
  • OpenCV & NumPy: Image preprocessing
  • Matplotlib: Visualization and Grad-CAM heatmaps

Impact: This AI-powered approach significantly speeds up flood assessment compared to manual methods, supporting rapid disaster response.
